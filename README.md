# Hospital Billing System

## Description
This project is a simple Hospital Billing System developed using the C programming language. 
The program calculates the total hospital bill based on room charges, medicine charges, 
and doctor consultation fees.

It also checks whether the payment made by the patient is sufficient and displays 
either the return amount or the remaining balance.

## Features
- Accept patient name
- Enter room charges
- Enter medicine charges
- Enter doctor fee
- Calculate total bill
- Verify payment status
- Display return amount or remaining balance

## Technologies Used
- Programming Language: C
- IDE: Dev-C++
- Version Control: GitHub

## How to Run
1. Open the program in Dev-C++.
2. Compile the program using a C compiler.
3. Run the program.
4. Enter patient details and charges.
5. The program will calculate and display the hospital bill.
